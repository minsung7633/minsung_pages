# 민성산업 홈페이지

정적 HTML/CSS로 구성된 민성산업 홈페이지입니다. GitHub Pages에서 루트 정적 사이트로 배포됩니다.

## 배포

`main` 브랜치에 push하면 `.github/workflows/pages.yml` 워크플로우가 실행되어 GitHub Pages로 배포합니다.

처음 연결할 때는 GitHub 저장소의 `Settings > Pages`에서 `Build and deployment` 소스를 `GitHub Actions`로 설정하세요.

## 로컬 확인

브라우저에서 `index.html`을 직접 열어 확인할 수 있습니다.
